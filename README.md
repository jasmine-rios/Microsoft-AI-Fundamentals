# Microsoft-AI-Fundamentals
Repo for study notes for Microsoft AI Fundamentals exam

It will include git repos, labs, notes, and other things that helped me study for Microsoft AI fundamentals

## WIP Indefinately

This repo is a work in progress as I learn more AI fundamentals.

This repo was started in November 2025 and names of services might change or consolidate into others.

"Azure AI Fundamentals (AI-900) Study Guide" by Tom Tauli **recommends** these certifications to build a better base for Microsoft AI but is **NOT REQUIRED for AI-900 study**:

- AZ-900 Microsoft Azure Fundamentals: Ideal for cloud services and how Microsoft delievers through Azure if you are new to Cloud

- DP-900 Microsoft Azure Data Fundamentals: Ideal if interested in the data side and learn core data processes and concepts (relational and nonrelational data, data analytics, data, data analytics)
